# Actividad-2.5-Proyecto-Panel-De-Control-Con-Bootstrap
Repositorio para almacenar actividad.
Crea un diseño para un panel de control (libre) para administrar el backend de una aplicación web utilizando Bootstrap.

El proyecto debe tener una página principal y varias secciones donde estén las interfaces para administrar las diferentes secciones de la web (crear posts, administrar grupos, administrar usuarios… elige algunas, no todas).

No utilices CSS. Aprovecha todas las clases que posee Bootstrap.

Intenta aplicar todos los contenidos vistos en las actividades anteriores y aplicar algunos nuevos consultando la documentación.
